DATA PROCESSING


Data processing is the process of collecting raw data, changing it into useful information, and presenting it in a form that people can understand and use.


Stages:


  1. Input: Collecting raw data from sources.

  2. Processing: Manipulating data into useful information.

  3. Storage: Saving data for future use.

  4. Output: Presenting processed data as information.

  5. Feedback: Using output information to influence further input.


Methods:


  • Manual processing

  • Mechanical processing

  • Electronic processing (using computers)


Applications:

  • Banks processing transactions.

  • Schools managing student records.

  • Businesses tracking sales.
