LOGIC GATE


Introduction:


Logic gates are electronic devices that perform logical operations on one or more binary inputs to produce a single output.



Basic Logic Gates:


  1. AND Gate: Output is 1 only if all inputs are 1.

  2. OR Gate: Output is 1 if at least one input is 1.

  3. NOT Gate: Output is the inverse of the input (also called inverter).


Truth Tables:


A B AND OR NOT A
0 0 0 0 1
0 1 0 1 1
1 0 0 1 0
1 1 1 1 0



Applications:


  • Used in building digital circuits and computer processors.

  • Basic building blocks for arithmetic and decision-making in computers.
