CORRECT SPACING AND PUNCTUATION MARKS




Correct spacing and punctuation marks are essential for clear and accurate writing. Here is how to use them properly:


  1. Period (.): Use one space after a period at the end of a sentence. Example: “It’s sunny today.”

  2. Comma (,): Use one space after a comma. Example: “I bought apples, bananas, and oranges.”

  3. Question Mark (?): Use one space after a question mark. Example: “How are you?”

  4. Exclamation Mark (!): Use one space after an exclamation mark. Example: “Wow! That’s great!”

  5. Colon (:): Use one space after a colon. Example: “Items needed: pen, paper, and a ruler.”


CORRECT FINGER PLACEMENT



Correct finger placement on a keyboard helps with efficient typing and comfort.


  1. Home Row : Left hand on A, S, D, F; right hand on J, K, L, ;.

  2. Thumbs : Use thumbs for the Spacebar.

  3. Enter Key : Use the right pinky finger.

  4. Backspace Key : Use the right pinky finger.

  5. Tab Key: Use the left pinky finger.

  6. Shift Keys: Use the opposite pinky finger (left for right shift, right for left shift).

  7. Movement: Use index, middle, and ring fingers to reach other keys while keeping home row fingers in place.
