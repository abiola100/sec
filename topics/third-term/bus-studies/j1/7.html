PART OF COMPUTER KEYBOARD




parts of computer key and their uses

  1. Alphanumeric keys : they are the primary keys used for typing letters (A-Z), numbers (0-9) and standard punctuation marks.

  2. Special function keys (F1-F12) : can be found on the top of the keyboard and serve specific purposes depending on which software you are using.

  3. Modifier Keys: These are keys such as Shift, Ctrl or Control, Alt - Alternate and Command (in Mac) that change the function of other keys when pressed together.

  4. Navigation keys: These include Arrow keys, Home key, End key, Page Up key and Page Down key that enable cursor movement within a text or document.

  5. Numeric keypad: A group of keys located at right side which resembles calculations in calculators are meant for easy number entry.

  6. Special Keys: These include buttons like Enter button or Spacebar button or Backspace button, Tab key, Esc or Escape button and Delete button that perform certain functions while typing and navigating.



SIMILARITIES BETWEEN TYPEWRITERS AND COMPUTER KEYBOARD


  1. QWERTY Layout: Mostly, both devices depend on the QWERTY key layout for alphanumeric keys which makes it easy for them to share typing experience.

  2. Typing Mechanism: Both apply pressing of keys as a means to insert letters, digits or symbols, having people translating physical key presses into characters on paper (typewriter) or on a screen (computer).

  3. Basic Key Functions: They have similar basic keys such as letters, numbers, punctuation marks and space bar for both typing and formatting text.

  4. Shift Key Usage: The upper characters found on some keys including punctuation marks are accessed using the Shift key while capital letters are typed with that same key.

  5. Manual Input: It is essential for an individual user to manually press keys so as they bring about actual production of texts (with each key representing its own character/function).
