DATABASE


A database is a collection of related data or information that is stored together in an organised way so that it can be easily accessed, managed, and updated.

In simple terms, a database is where information is kept for quick retrieval, use, and management.

Types of Databases


  1. Manual Database : A database that is created and managed by hand without using a computer. Example: Record books, index cards, filing cabinets.

  2. Electronic (Computerised) Database : A database that is created and managed using a computer and software. Example: Microsoft Access, Oracle, MySQL, Excel (as a simple database).

Characteristics of a Good Database


  1. It stores data in an organised way.

  2. It allows easy access to data.

  3. It makes updating and deleting data simple.

  4. It allows sharing of data among users.

  5. It ensures accuracy and reduces errors.


Advantages of Electronic Databases


  1. Information can be found quickly.

  2. Data can be easily updated or corrected.

  3. It saves space ‚Äî no need for bulky files and cabinets.

  4. It allows sharing of data between different departments.

  5. It Helps to protect data with passwords and security features.


disadvantages of electronic databases


  1. Requires electricity and computers to work.

  2. May be costly to set up and maintain.

  3. Risk of data loss due to technical failure or hacking.

  4. Needs skilled personnel to manage and operate the system.

  5. Vulnerable to cyberattacks and unauthorized access if not well secured.
