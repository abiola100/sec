GRAPHICS PACKAGES
Meaning of Graphic Packages

A graphic package is a computer software that allows the user to create, edit, and manipulate drawings, pictures, and images on the computer.

It helps you to produce different types of visual work such as posters, charts, logos, diagrams, banners, and greeting cards.

A graphic package is mainly used for designing and illustrating information in a visually appealing way.



Examples of Graphic Package
  1. Microsoft Paint

  2. CorelDRAW

  3. Adobe Photoshop

  4. PaintShop Pro

  5. Harvard Graphics

  6. Lotus Freelance Graphics

  7. Print Artist

  8. Adobe Illustrator

  9. GIMP (GNU Image Manipulation Program)

  10. Inkscape



Uses of Graphic Packages
  1. To create and edit drawings and pictures.

  2. To design posters, banners, and handbills.

  3. To create charts, graphs, and diagrams.

  4. To enhance pictures and photographs.

  5. To produce logos, business cards, and letterheads.

  6. To design advertisements and magazines.

  7. To make animations and illustrations.

  8. To teach art and design in schools.



Features (Functions) of Graphic Packages
  1. Menu Bar – Contains commands such as File, Edit, View, etc.

  2. Tool Box or Tool Bar – Contains drawing tools like pencil, brush, eraser, line, and shapes.

  3. Color Palette – Provides colors for filling or outlining objects.

  4. Work Area / Drawing Area – The main area where drawings and designs are created.

  5. Title Bar – Displays the name of the program and the current file.

  6. Status Bar – Shows information about the current tool or position of the cursor.

  7. Scroll Bars – Help you to move around the work area.

  8. Zoom Tool – Used to enlarge or reduce the view of a drawing.

  9. Text Tool – Allows you to insert and format text.

  10. Shape Tools – Help to draw geometric figures like circles, rectangles, and polygons.



Examples of Tools Found in a Graphic Package (Microsoft Paint)
  1. Pencil Tool – Used for freehand drawing.

  2. Brush Tool – Used for painting or shading.

  3. Eraser Tool – Used to remove unwanted parts of a drawing.

  4. Fill with Color Tool – Used to apply color to an area.

  5. Text Tool – Used to add text to a design.

  6. Line Tool – Used to draw straight lines.

  7. Rectangle Tool – Used to draw rectangles or squares.

  8. Ellipse Tool – Used to draw circles and ovals.

  9. Color Picker Tool – Used to select and copy colors from an image.

  10. Magnifier Tool (Zoom) – Used to enlarge part of a drawing.



Steps for Starting Microsoft Paint
  1. Click on the Start Menu.

  2. Select All Programs (or All Apps).

  3. Click on Accessories.

  4. Select and click on Paint.

  5. The Paint window will open, ready for use.



Parts of Microsoft Paint Window
  1. Title Bar – Displays the name of the program.

  2. Menu Bar – Contains options like File, Edit, and View.

  3. Tool Box – Contains drawing tools.

  4. Color Palette – Shows color options.

  5. Drawing Area – The white space where you create designs.

  6. Scroll Bars – Used to move around the work area.

  7. Status Bar – Displays information about the drawing.



Creating a Drawing in Microsoft Paint
  1. Open Microsoft Paint.

  2. Select a drawing tool (e.g., Pencil or Line Tool).

  3. Draw the desired object on the Drawing Area.

  4. Use the Color Tool to fill the object with color.

  5. Add Text if necessary using the Text Tool.

  6. Save your drawing using the File → Save As option.



How to Save a Drawing
  1. Click File on the Menu Bar.

  2. Select Save or Save As.

  3. Type the file name.

  4. Choose the location (e.g., Desktop, Documents).

  5. Click Save.



How to Open a Saved Drawing
  1. Click File on the Menu Bar.

  2. Select Open.

  3. Choose the file you saved earlier.

  4. Click Open to display it.



How to Print a Drawing
  1. Click File on the Menu Bar.

  2. Select Print.

  3. Choose your printer.

  4. Click OK to print.



Advantages of Graphic Packages
  1. They make drawing and designing easy and fast.

  2. They allow correction of mistakes easily.

  3. They produce neat and accurate drawings.

  4. They can store drawings for future use.

  5. They allow coloring, shading, and editing of pictures.

  6. They are useful in advertising and education.

  7. They make professional designs possible without manual effort.



Disadvantages of Graphic Packages
  1. They require a computer and electricity to work.

  2. Some are expensive to buy and install.

  3. They may need special training to use effectively.

  4. They can cause eye strain or fatigue if used for too long.

  5. Files may be lost or damaged if not saved properly.



Uses of Graphic Packages in Daily Life
  1. In schools – for making charts and illustrations.

  2. In businesses – for designing logos and advertisements.

  3. In printing – for making posters, handbills, and banners.

  4. In entertainment – for designing animations and games.

  5. In architecture – for drawing building plans.

  6. In fashion – for designing clothes and patterns.



Examples of Activities in Graphic Packages
  1. Drawing shapes (lines, circles, rectangles).

  2. Coloring drawings.

  3. Adding text to pictures.

  4. Editing images.

  5. Designing greeting cards or banners.

  6. Creating charts and diagrams.

  7. Making simple animations.





Summary

  1. A graphic package is software used to create and edit drawings and images.

  2. Common examples are Paint, CorelDRAW, and Adobe Photoshop.

  3. It contains tools such as Pencil, Eraser, Text, and Color Fill.

  4. It is useful for designing, illustrating, and editing visual materials.

  5. Proper use of graphic packages encourages creativity and accuracy.
