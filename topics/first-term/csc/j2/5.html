CONVERSION OF ONE UNIT TO ANOTHER



  • 1 Bit (b) = 1/8 Byte (B)

  • 1 Byte (B) = 8 Bits (b)

  • 1 Kilobyte (KB) = 1,024 Bytes (B)

  • 1 Megabyte (MB) = 1,024 Kilobytes (KB)

  • 1 Gigabyte (GB) = 1,024 Megabytes (MB)

  • 1 Terabyte (TB) = 1,024 Gigabytes (GB)



differences between bit, byte, kilobyte, megabyte, gigabyte, and terabyte


  1. Bit (b):

    • Smallest unit of data.

    • Represents a binary value: 0 or 1.

    • Used in data transfer rates (e.g., internet speed).

  2. Byte (B): Stores one character of data (e.g., a letter or number).Common unit for measuring file sizes.

  3. Kilobyte (KB): Used to measure small files like text files and low-resolution images.

  4. Megabyte (MB): Used for medium-sized files like audio files, high-resolution images, and documents.

  5. Gigabyte (GB): Used for larger files like videos, games, and apps.

  6. Terabyte (TB): Used for massive storage needs like hard drives, data centers, or cloud storage.
