COMPUTER SOFTWARE
What is Software?

Software is the set of instructions, programs and data that tell the computer what to do.

Whereas hardware refers to the physical parts you can touch (keyboard, monitor, etc.), software refers to the non-physical parts (programs, applications) that run on the hardware.

According to the curriculum for Computer Studies in Nigeria, the study of software is listed under “Computer Software” alongside hardware.

So: Software = instructions + data + programs.



Why is Software Important?
  1. It helps you to make the computer useful. Without software, a computer is just hardware with no functionality.

  2. Software enables you to run applications like word-processing, spreadsheets, games, internet browsers, etc.

  3. It helps you to interact with the computer hardware in a meaningful way.

  4. It helps you to carry out tasks more efficiently — from writing documents to drawing pictures to playing music.

Understanding software is important because you will use many types of software in your school computer-lab lessons.



Types of Software

Software can be categorised in different ways. The main two categories are: System Software and Application Software.

1 System Software

System software is the software that helps the computer to manage its internal operations and hardware.

Examples include Operating Systems (OS), device drivers, utility programs.

The OS is the main piece of system software — it controls memory, devices (input/output), files, etc.

For example, when you switch on the computer (booting), the operating system loads and prepares the computer for use.

System software acts between the hardware and the applications you use.

2 Application Software

Application software is the software you use to perform specific tasks.

Examples: word processor (for typing documents), spreadsheet (for managing numbers), presentation software (for slides), web browser (for internet), games, graphic design software.

These run on top of the system software (OS).

Other Possible Sub-types / Classifications

  1. Utility Software: A subset of system software that helps you maintain the computer (e.g., antivirus, disk defragmenter, backup software).

  2. Programming Software: Tools used to create software (compilers, interpreters, editors).

  3. Middleware: Software that connects different applications or systems (less common in JSS level).

  4. Firmware: Software embedded into hardware devices.



Examples of Software
  1. Operating Systems: e.g., Windows, Linux, macOS .

  2. Word Processor: e.g., Microsoft Word, LibreOffice Writer.

  3. Spreadsheet: e.g., Microsoft Excel, LibreOffice Calc.

  4. Presentation Software: e.g., PowerPoint.

  5. Web Browser: e.g., Google Chrome, Mozilla Firefox.

  6. Utility: e.g., antivirus program, disk cleaner.

  7. Desktop Publishing: e.g., software to combine text and graphics for print.

  8. Database Software: e.g., Microsoft Access or simple database programs.



Functions / What Software Does

Here are what software does for you

  1. it helps you to Manage hardware resources (via system software) — allocate memory, control input/output devices.

  2. it helps you to Provide a user interface — allow you to interact with the computer (e.g., via desktop icons, menus).

  3. it helps you to Run applications — such as processing documents, spreadsheets, browsing the internet.

  4. it helps you to Facilitate tasks — e.g., calculation, data storage, presentation, communication.

  5. it helps you to repetitive work — e.g., using templates, macros in spreadsheets.

  6. it helps you to Ensure security and maintenance — via antivirus, backups, system updates (utility software).

  7. it helps you to Store and retrieve data — via database software or file systems.

  8. it helps you to Enable communication — software supports internet, networking, messaging.



Distinguishing Between Software and Hardware
  1. Hardware: physical parts you can touch — monitor, keyboard, CPU, hard-disk.

  2. Software: intangible instructions, programs, data — cannot be touched.

Example: The keyboard is hardware; the program you type in (e.g., word processor) is software.

The hardware needs software to do anything useful, and software runs on hardware.



Some Key Terms You Should Know
  1. Booting: Starting up a computer; when you switch on the computer and the system software (OS) loads.

  2. Operating System (OS): The main system software that manages computer hardware and software resources.

  3. Driver: A small piece of software that allows the OS to communicate with hardware devices.

  4. Application: Software designed to help the user to perform tasks.

  5. Utility: Software used for maintenance or optimisation of the computer.

  6. Installation: The process of copying software onto a computer to make it usable.

  7. File Management: Organising files into folders, saving, moving, deleting – often done via software.

  8. Program vs Application: A program may be a small set of instructions; an application is a complete software tool for user tasks.

  9. Update: Software improvements or patches (often for security or bug-fixes).

  10. License: The legal permission you have to use software (free, paid, open source).

  11. Open Source Software: Software whose source code you can access, modify and distribute (not always in JSS level, but good to know).



Activities / Questions (for Practice)
  1. Define software.

  2. Differentiate between system software and application software.

  3. Give three examples each of system software and application software.

  4. What is the role of an operating system?

  5. Why is software important for computers?

  6. Explain what utility software is and give one example.

  7. What is the difference between installation and update of software?

  8. Describe what a driver does in a computer system.

  9. List four tasks you can do with application software.

  10. How does software help you to manage files on your computer?





Summary

In summary:

Software is a collection of programs, instructions and data which direct the operations of a computer.

It is essential because without it, the computer hardware cannot perform any useful function.

Software is broadly divided into system software (which helps to run the computer itself) and application software (which helps users to do tasks).

You should be familiar with examples of each type, understand their functions, and know key terminology.

This topic forms a foundational part of the Junior Secondary School Computer Studies curriculum in Nigeria, so mastering it will help you in subsequent topics (like hardware, file management, programming, etc.).
