INTERNET SEARCH ENGINE


An Internet search engine can be defined as a program that helps the users to find information on the internet by showing the related websites. It works like a helper or guide that searches the web when you type a word or question.



Examples of Internet Search Engines


  1. Google (www.google.com)

  2. Bing (www.bing.com)

  3. Yahoo Search (www.yahoo.com)

  4. DuckDuckGo (www.duckduckgo.com)

  5. Ask.com (www.ask.com)


Functions of a Search Engine


  1. Find information on any topic.

  2. It show related websites that contain the information you want.

  3. internet search engine Help you locate images, videos, maps, and news.

  4. it Provide quick answers to simple questions.

  5. it Suggest related search topics.



How a Search Engine Works


  1. The user types a word, phrase, or question (called a search query) in the search box.

  2. The search engine looks through millions of websites very quickly.

  3. It brings out a list of results (links) that match what the user typed.

  4. The user clicks on one of the links to visit a website and read the information.


Parts of a Search Engine Page


  1. Search box – Where you type your question or keyword.

  2. Search button – The button you click to start the search.

  3. Search results – The list of websites related to your search.

  4. Links – The clickable titles that take you to a website.


Uses of Search Engines


  1. To do homework and research.

  2. To find pictures, videos, and maps.

  3. To read news and current events.

  4. To learn new skills (e.g., how to draw, cook, or play a game).

  5. To get answers to simple or complex questions.
